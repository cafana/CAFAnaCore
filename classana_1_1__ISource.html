Detailed Description

template<class RecT>
class ana::_ISource< RecT >

Simple base class for various RecordSources etc.

A source has a list of sinks with the same RecT to which it will send its events / livetime / etc. It provides methods to allow the sink to move or unregister itself. All the detailed implementation of sources happens in derived classes.
